TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Issue: Backend won't start
  Solution: Port 8001 may be in use
  Check: netstat -ano | findstr 8001
  Kill:  taskkill /PID <pid> /F

Issue: Tests failing
  Solution: Check NLP models
  Run:   python diagnose_spacy.py

Issue: Models not found  
  Solution: Download manually
  Run:   python -m spacy download en_core_web_sm

Issue: Can't find Python
  Solution: Add Python to PATH or use absolute path
