Model Card — AM-DS-01
======================

Overview
--------
- Purpose: Rank candidates and flag recruiter-ready profiles.
- Models:
  - Model A (Regression): Predicts `AI Score (0-100)`.
  - Model B (Classification): Predicts `Recruiter_Decision` probability.
- Data: `data/enriched_dataset.csv` (engineered features from resumes)
- Splits: 5-fold GroupKFold by `Resume_ID` to prevent leakage.

Training & Features
-------------------
- Seed: 42
- Features: see `pipeline_config.json` (`features_used`)
- Imbalance handling (CV): `class_weights` selected; benchmark compared undersample/oversample.
- Winsorization: used for robust numeric handling.
- Categorical: No non-binary categorical columns found; binary booleans retained.

Metrics (CV)
------------
- Model A (AI Score): MAE 6.018, RMSE 8.133, R² 0.847; Pearson 0.946; Spearman 0.884.
- Model B (Recruiter): ROC AUC 0.9976; PR AUC 0.99947; F1 0.9902; Precision 0.9866; Recall 0.9938.
- Calibration (Model B): Mean Brier improved 0.03135 → 0.02711.

Intended Use
